# AWS Project 1 - Static Website Hosting

This is my hands-on AWS cloud project where I built and deployed a static website.

## 🚀 Project Overview
In this project, I hosted a static website using Amazon S3 and distributed it globally using Amazon CloudFront with HTTPS enabled.

## 🧰 Technologies Used
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront (CDN)
- HTML, CSS, JavaScript

## 🌍 Live Demo
👉 https://d1zg6j5dy4ggmp.cloudfront.net

## 📂 Project Structure
- `index.html` – Main webpage
- `style.css` – Styling
- `script.js` – JavaScript functionality

## 🔐 Key Features
- Static website hosting on AWS
- Secure HTTPS delivery
- Fast global content delivery with CloudFront
- Simple frontend interaction (JavaScript)

## 📚 What I Learned
- How to host a static website using S3
- How to configure CloudFront distribution
- How to enable HTTPS for a website
- Basic AWS deployment workflow
